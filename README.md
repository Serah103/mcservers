## 🧰 What mcservers does

mcservers is a simple server.dat editor for Minecraft on Windows.

It helps you view and edit the list of saved multiplayer servers that Minecraft uses. If you want to add a server, remove one, or change a server name or address, this tool gives you a simple way to do it.

## ✨ What you can do

- Open a Minecraft `servers.dat` file
- View saved servers in a clear list
- Add a new server entry
- Edit a server name or address
- Remove a server you no longer use
- Save changes back to the same file
- Work with Minecraft server lists without editing files by hand

## 🗃️ Find your Minecraft servers file

Minecraft stores server data in a file called `servers.dat`.

You can usually find it in your Minecraft profile folder. On Windows, it is often inside a path like this:

`C:\Users\YourName\AppData\Roaming\.minecraft`

If you do not see the folder, make sure hidden items are shown in File Explorer.

## 📝 How to edit your server list

Once mcservers is open:

1. Load the `servers.dat` file
2. Wait for the server list to appear
3. Select a server you want to change
4. Edit the name or address fields
5. Save the file when you are done

If you want to add a new server, use the add option in the app and fill in the server name and address.

If you want to remove a server, select it and delete it from the list.

## 📌 Tips for safe editing

- Close Minecraft before you edit `servers.dat`
- Keep a backup copy of the file before you change it
- Save after each change if you are editing many servers
- Use the exact server address your host gave you
- Check for extra spaces in names and addresses
